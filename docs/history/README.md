# Compiler History Archive

This directory contains completed drive notes, phase summaries, and audit snapshots
that are useful for historical context but are not active planning sources.

Active entry points remain at the repository root:

- `README.md` for project orientation
- `ROADMAP.md` for the current compiler planning surface
- `CLAUDE.md` for local agent instructions

Do not use archived ROADMAP variants as current task state without checking the
root `ROADMAP.md` and the top-level workspace roadmap first.
