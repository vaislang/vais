# Vaisプログラミング言語

**Vais**は、最大限のトークン効率と開発者の生産性を目指して設計された、AI最適化システムプログラミング言語です。

## 主な特徴

- **単一文字キーワード**: 関数は`F`、構造体は`S`、条件分岐は`I`、パターンマッチは`M`
- **式指向**: すべてが値を返す
- **自己再帰演算子**: `@`で簡潔な再帰関数を記述
- **高度な型システム**: 完全な型推論、ジェネリクス、トレイト
- **LLVMバックエンド**: 最適化されたコード生成によるネイティブパフォーマンス
- **モダン機能**: async/await、パターンマッチング、`?`と`!`によるエラーハンドリング

## Vaisを選ぶ理由

VaisはAI生成コードのトークン使用量を最小化しながら、システムプログラミングの完全な能力を維持します:

- **Rust/C++と比較して50-70%少ないトークン**
- **セルフホスティングコンパイラ** 50,000行以上のVaisコード
- **774K行/秒** のコンパイル速度
- **2,500以上のテスト** すべてのコンポーネントに対して

## 例

```vais
# 自己再帰によるフィボナッチ
F fib(n: i64) -> i64 {
    I n <= 1 { R n }
    @(n-1) + @(n-2)
}

F main() {
    puts("fib(10) = ")
    print_i64(fib(10))
}
```
