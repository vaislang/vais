This document outlines the comprehensive social media strategy for **Vais**, an AI-optimized systems programming language. Vais is designed to minimize token usage while maximizing code expressiveness, making it ideal for AI-assisted development and LLM code generation.

### About Vais

**Vais** is characterized by:
- **Single-letter keywords** (`F`, `S`, `E`, `I`, `L`, `M`) for token efficiency
- **Self-recursion operator** `@` for elegant recursive calls
- **LLVM backend** delivering native performance comparable to Rust and C
- **Rust-like safety** with comprehensive type system
- **AI-friendly syntax** designed for LLM code generation and AI assistance

## 3. Tone & Voice Guidelines

### Core Principles

1. **Technical but Accessible**
   - Explain complex concepts in understandable terms
   - Avoid unnecessary jargon, but don't oversimplify
   - Use examples that resonate with both beginners and experts

2. **Enthusiastic & Positive**
   - Show genuine excitement about language features
   - Celebrate community contributions and achievements
   - Maintain an optimistic tone about the project's future

3. **Educational & Helpful**
   - Position Vais as a learning resource
   - Provide actionable tips and best practices
   - Link to detailed documentation for deeper learning

4. **Professional & Respectful**
   - Respect other programming languages and communities
   - Avoid negative comparisons or dismissive language
   - Acknowledge Vais's position in the broader ecosystem

### Tone Examples

**DO âœ…**
- "Vais makes recursive functions elegant with the @ operator"
- "Here's a pattern that works great in Vais..."
- "Our community built something amazing!"
- "Curious about token efficiency? Here's why it matters..."
- "Check out this benchmarkâ€”we're competitive with Rust!"

**DON'T âŒ**
- "Vais is better than everything else"
- "Other languages don't understand token efficiency"
- "Why would you use [language] when you have Vais?"
- "We're the only language that gets it right"
- Passive-aggressive comparisons or sarcasm

### Linguistic Style

- **Use active voice** when possible
- **Keep sentences concise** for social media
- **Use emojis strategically** (1-2 per post on Instagram, 1-3 on Twitter)
- **Create visual hierarchy** with bullet points and formatting
- **Include CTAs** (Calls-to-Action) naturally in each post
