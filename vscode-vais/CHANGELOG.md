# Change Log

All notable changes to the "vais" extension will be documented in this file.

## [0.1.0] - 2026-01-19

### Added
- Initial release of Vais language support
- Syntax highlighting for all Vais keywords (F, S, E, I, L, M, W, T, X, V, C, R, B, N, A)
- Support for primitive types (i8, i16, i32, i64, u8, u16, u32, u64, f32, f64, bool, str)
- Highlighting for self-recursion operator `@`
- Async/await keyword support
- String and numeric literal highlighting
- Comment support with `#`
- Bracket matching and auto-closing pairs
- Smart indentation rules
- Code folding support

### Features
- TextMate grammar for comprehensive syntax highlighting
- Language configuration for VSCode editor features
- Support for operators: arithmetic, comparison, logical, bitwise, assignment
- Built-in function highlighting (puts, putchar, print_i64, malloc, free, strlen)
- Pattern matching syntax support
- Trait and implementation syntax highlighting
