Vais

Vais is a .vais language and self-host compiler workspace. The public compiler command is scripts/vaisc.

Run Vais Read Docs

Quick start

Compile a verified .vais program.

The public compiler command builds a native host driver and uses clang to link generated LLVM IR. 

git clone https://github.com/vaislang/vais
cd vais

scripts/vaisc doctor
scripts/vaisc run examples/c4.vais
scripts/vaisc emit-ir examples/c4.vais -o /tmp/c4.ll
scripts/vaisc build examples/c4.vais -o /tmp/c4

scripts/install-vaisc.sh --prefix "$HOME/.local"
"$HOME/.local/bin/vaisc" doctor

Language surface

Current Vais uses full-word syntax.

Public examples are limited to the syntax protected by the compiler gates. The main scalar type is Int, variable binding uses let, and control flow uses if, else, and while.

Structs

# expect: 42
struct Box {
    value: Int,
}

fn main() -> Int {
    let b = Box { value: 42 }
    return b.value
}

Enums and match

# expect: 2
enum Color { Red, Green, Blue }

fn color_number(c: Color) -> Int {
    match c {
        Color.Red => return 1,
        Color.Green => return 2,
        Color.Blue => return 3,
    }
}

fn main() -> Int {
    let c = Color.Green
    return color_number(c)
}

Loops

# expect: 10
fn main() -> Int {
    let mut s = 0
    let mut i = 0
    while i < 5 {
        s = s + i
        i = i + 1
    }
    return s
}

Compiler

One public command: scripts/vaisc.

The compiler accepts .vais files, emits LLVM IR, and links native binaries through clang. Normal emit-ir, build, and run commands use the native public driver.

Full engine

Runs through a native host driver linked with the reusable self-host compiler core checked in at compiler/self/vaisc_core.ll.

Native command

scripts/vaisc doctor checks the local compiler path and reports the linked self-host core.

Direct path

--engine direct stays native for Int helpers, locals, calls, if, while, simple struct helpers, List<Int> local, list/element assignment, parameter, inline, return, returned-argument, plain if-condition, and loop-hoisted operations, plus List<Struct> construction with [], list(), list literals, list/element assignment, push, len, index, field reads/writes, parameter/return ABI, inline arguments, and returned-argument hoisting in statements plus plain if and while conditions.

Standalone install

scripts/install-vaisc.sh installs a standalone vaisc binary outside the checkout.

Release archives

Tag builds publish standalone compiler archives through the release archive workflow.

Self-host status

compiler/self/*.vais contains the compiler work written in Vais and is protected by self-host fixpoint gates.
